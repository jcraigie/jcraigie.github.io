Classes versus ID's and how to use them. So you've come along to styling some html with css and you have an element, such as a paragraph, you want to apply some changes to but there are other paragraph's in the code, what do you do? you give that paragraph a unique identifier via a class or an id. it's as simple as:


A class would be used when you want apply that styling to multiple elements, not neccesarily of all the same tag type either. An id would be used when you want to apply the styling to that one singular element. Essentially if there is a possibility you would want to apply the styling in question to any other elements you would use class and id if it is just a one off thing for that element and no others. Keep in mind that with the cascading nature of css you can apply multiple sets of styling so placement matters in the stylesheet.
