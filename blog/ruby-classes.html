Making the real virtual.

Classes can be used to model real world objects that can then be interacted with or manipulated by other code. The physical traits like color, height, fuel, etc. can be represented by variables. And methods can be used to represent behaviors of the object. So a car would have its fuel variable set to gasoline and an accelerator_pedal method that results in the car accelerating.
