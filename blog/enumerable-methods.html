Map

1/31/2016

Map is a Ruby Enumerable that is similar to .each in that it iterates through an array but it returns the values in a new array after running any code present in the code block. Some think of it as “mapping” the code block to each value of the array before returning the values. For Example:

Returns [1, 4, 9, 16, 25]


This provides the user with a nicely streamlined way to iterate over an array with their code and returning it to a new array.
